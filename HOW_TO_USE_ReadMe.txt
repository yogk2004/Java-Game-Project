To Run Our Project System should have JavaFx installed 

	first extract the Project folder from zip and open it in Intellij IDEA 

	now go to src\main\java\com\example\stickherogame_ap_project_deadline1
	
	right click on the file Main.java and select option Run "Main.main()"

output will be javaFx Screen and you have to click on play button to move to next screen

second screen is the initial UI of the level of Game

Exit button will take you to end screen and here you can click on score to exit the Game.
